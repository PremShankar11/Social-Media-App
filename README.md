# IP Mini Project

Friends-first social app focused on real connections instead of addictive engagement patterns.

## Approved stack

- `Vite`
- `React`
- `TypeScript`
- `Zustand`
- `Tailwind CSS`
- `Supabase`
- `Zod`

## Current foundation goal

Build the project foundation before feature development:
- clean app scaffold
- docs and scope tracking
- Supabase-ready folder structure
- initial schema and RLS planning

## Workspace structure

```text
IP_mini_project/
  foundation.md
  README.md
  .env.example
  app/
  docs/
  supabase/
```

## Next milestones

1. Finish the foundation setup
2. Connect Supabase
3. Create the first schema migration
4. Build auth and profile flow

## Local setup note

When we connect Supabase, place frontend environment variables in `app/.env`.
Use `app/.env.example` as the starting template.
