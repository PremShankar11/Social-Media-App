# Product Scope

## Product summary

A private-by-default social app where users primarily connect with accepted friends, share photos or videos, and consume a chronological feed without algorithmic manipulation.

## Target users

- students
- friend groups
- small communities that want a less noisy social experience

## MVP goals

- secure sign up and login
- create and edit user profile
- send and accept friend requests
- create text/image/video posts
- upload compressed media
- view a friends-only chronological feed
- receive basic notifications

## Explicitly out of scope for foundation

- public creator tools
- recommendation algorithms
- streak systems
- marketplace features
- ad systems
- advanced analytics
